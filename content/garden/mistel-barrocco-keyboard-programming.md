Because I tend to accidentally enter macro programming mode and mess up my macro.

To program a macro, for example `fn+L` -> `Win+L`:

1. Press `Fn + R_Ctrl` to enter marco mode. (LED1 will turn solid white)
Ctrl
2. Press `Fn + L` (LED1 will blink white)
3. Press `Win+L`
4. Press the `Pn` key once to save the new setting. LED1 will turn solid white again.
5. Press `Fn + R_Ctrl` again to exit the macro mode. LED1 will turn off.
